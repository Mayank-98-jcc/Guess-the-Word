# Imposter Who?

A local word-based party game built with React, Vite, Tailwind CSS, Framer Motion, React Router, and Context API.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

## Flow

1. Add players on the setup screen.
2. Choose a category, number of imposters, and optional time limit.
3. Pass the phone and reveal each player's card privately.
4. Discuss together and try to expose the imposter.
5. Reveal the answer and start a new round.
